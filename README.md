# 🏒 Air Hockey Game

A fast-paced, arcade-style Air Hockey game built using **HTML5 Canvas, CSS, and Vanilla JavaScript**. Challenge an AI opponent in a futuristic neon arena featuring dynamic physics, particle effects, slow-motion game points, sound effects, speed boosts, confetti celebrations, and smooth gameplay.

---

## 🎮 Features

### 🏒 Core Gameplay

* Player vs AI Air Hockey
* Realistic puck movement and collision physics
* Mouse and touch controls
* Goal detection and score tracking
* Game over screen with winner announcement

### ⚡ Dynamic Difficulty

* Puck speed increases every 2 goals
* Progressive match intensity
* Game Point slow-motion cinematic effect
* AI opponent with prediction and reaction mechanics

### ✨ Visual Effects

* Neon-themed game arena
* Particle explosion effects
* Puck trails
* Screen shake on powerful hits
* Goal flash animations
* Victory confetti celebration
* Glow effects and dynamic lighting

### 🔊 Audio Effects

* Paddle hit sounds
* Wall collision sounds
* Goal scoring effects
* Speed-up notifications
* Victory celebration sound

### 📊 Match Statistics

* Goals scored
* Best scoring streak
* Top shot speed
* Power hit count
* Rally tracking

### 📱 Controls

* Mouse movement (Desktop)
* Touch controls (Mobile)
* Press **S** to mute/unmute sound
* Press **Space** to restart after game over

## 🎯 Gameplay Rules

1. Control the blue striker using your mouse or touch.
2. Score goals against the CPU opponent.
3. First player to reach the target score wins.
4. Every two goals, the puck speed increases.
5. At game point, the match enters cinematic slow-motion mode.
6. Win the match to trigger a confetti celebration.
